Welcome! I'm a dedicated Game Programmer with a strong passion for C++ programming, particularly in gameplay and networking.
Currently, I am immersed in two exciting projects that showcase my expertise and creativity.

Firstly, I'm working on Ember Sword, a web-based MMORPG developed using the Radiance engine.
Radiance is a custom C++ game engine built on an ECS framework, designed for optimal performance in both browser and client environments.

Secondly, I am developing my personal project, Farmion.
This is a time-management co-op farming game set in a magical wizarding world where players can explore, gather, craft, and build their own farms. Farmion is being brought to life using Unreal Engine 5.
